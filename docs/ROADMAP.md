# Roadmap

## Phase 0: Foundation

- Monorepo foundation.
- Deterministic local scanner.
- Transparent scoring.
- CLI and starter generators.
- Sample dashboard.
- Tests, docs, CI, and validation workflow.

## Phase 1: Local Scanner + Dashboard

- Better rule coverage.
- Richer dashboard states.
- Markdown and JSON report export.
- Configurable scan profiles.

## Phase 2: GitHub Action + PR Suggestions

- Score gates in CI.
- SARIF reporter.
- PR comments and annotations.
- Patch suggestions for instruction and docs improvements.

## Phase 3: Docs Crawler + OpenAPI/MCP Analysis

- Crawl documentation sites safely.
- Validate OpenAPI examples and operation descriptions.
- Inspect MCP tool schemas and descriptions.

## Phase 4: Agent Benchmark Runner

- Define realistic task suites.
- Run deterministic simulations first.
- Add optional agent execution adapters later.

## Phase 5: Hosted SaaS

- Project history.
- Team dashboards.
- Organization policies.
- Integration marketplace.
