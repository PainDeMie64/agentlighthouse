# Claude Project Memory: Sample Good Project

Sample Good Project is a small TypeScript API service for AgentLighthouse validation.

## Development Workflow

- Install with `pnpm install`.
- Run `pnpm lint`, `pnpm typecheck`, and `pnpm test` before handing work back.

## Testing Expectations

Core behavior should have Vitest coverage. Keep examples deterministic and local.

## Product Boundaries

This sample is not a production API, chatbot, or hosted SaaS. Do not build auth or billing here.
