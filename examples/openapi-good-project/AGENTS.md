# Agent Instructions

## Project Overview

This sample API project demonstrates an agent-readable OpenAPI contract.

## Setup Commands

```bash
npm test
```

```bash
npm run lint
```

```bash
npm run typecheck
```

## Architecture Map

- `openapi.yaml`: API contract.
- `agentlighthouse.tasks.yaml`: agent task benchmark.

## Coding Conventions

Keep examples placeholder-safe and update tests with contract changes.

## Common Mistakes

- Do not use real API keys.
- Do not edit generated build output.
- Do not remove error responses when simplifying examples.

## Security and Privacy

Never use real API keys. Use `EXAMPLE_API_KEY` in examples.

## Ownership

API contract changes require maintainer review.
