# v0.1.0-alpha.1 Release Report

Status: released and verified.

## Summary

`v0.1.0-alpha.1` is an incident recovery release for npm installability. It replaces the
broken CLI package metadata published in `0.1.0-alpha.0` and is the first usable npm alpha
for CLI consumers.

## Release Artifacts

- Release commit: `f07a7788338162d8ef5469d9fced3d27eca89895`
- Git tag: `v0.1.0-alpha.1`
- Tag status: annotated tag pushed to origin; resolves to the release commit.
- GitHub prerelease: https://github.com/PainDeMie64/agentlighthouse/releases/tag/v0.1.0-alpha.1

## npm Dist-Tags

Final verified dist-tags:

- `@agentlighthouse/core`: `alpha -> 0.1.0-alpha.1`, `latest -> 0.1.0-alpha.1`
- `@agentlighthouse/cli`: `alpha -> 0.1.0-alpha.1`, `latest -> 0.1.0-alpha.1`

## alpha.0 Deprecation

`@agentlighthouse/cli@0.1.0-alpha.0` is deprecated with:

> Broken alpha: workspace dependency leaked into published package metadata. Use
> @agentlighthouse/cli@0.1.0-alpha.1 or later.

## Post-Release Install Verification

Verification was run in a clean temporary npm consumer project outside the monorepo.

Verified commands:

```bash
npm init -y
npm install @agentlighthouse/cli@alpha
npx agentlighthouse --help
npx agentlighthouse version
npx agentlighthouse scan <sample-good-project>
npx agentlighthouse scan <sample-good-project> --format json --output scan.json
npx agentlighthouse baseline create <sample-good-project> --output baseline.json
npx agentlighthouse baseline validate baseline.json
npx agentlighthouse scan <sample-good-project> --baseline baseline.json --report-dir reports
npx @agentlighthouse/cli@alpha --help
npx @agentlighthouse/cli@alpha version
npm install @agentlighthouse/cli
npx agentlighthouse version
```

Results:

- `npx agentlighthouse version`: `0.1.0-alpha.1`
- `npx @agentlighthouse/cli@alpha version`: `0.1.0-alpha.1`
- Untagged `npm install @agentlighthouse/cli` resolves to `0.1.0-alpha.1`.
- Scan, JSON output, baseline create/validate, and report bundle generation passed.
